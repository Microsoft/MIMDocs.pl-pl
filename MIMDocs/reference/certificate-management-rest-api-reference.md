# <a name="certificate-management-rest-api-reference"></a>Dokumentacja interfejsu API REST zarządzania certyfikatami

Operacje zarządzania kartami inteligentnymi są również obsługiwane przez ten interfejs API. Interfejs API został opisany w następujących sekcjach:
