# <a name="management-agent-run-error-codes"></a>Kody błędów uruchom agenta zarządzania

Poniższe tabele zawierają kody błędów, które mogą być wyświetlane w interfejsie użytkownika Menedżera usługi synchronizacji w programie Microsoft Identity Manager (MIM) 2016, a także opisy dla każdego z tych błędów.

## <a name="connection-errors"></a>Błędy połączenia

| Błąd                  | Opis                                    |
|--------------------------|--------------------------------------------|
| nie powiodło się = połączenia         | Połączenie połączonego katalogu nie powiodło się z przyczyn innych niż uwierzytelniania. Na przykład sieć jest niedostępna lub serwer docelowy jest w trybie offline.                                                                                                                                                                                                                                                                                                                      |
| porzucone połączenia        | Połączenie między połączonego katalogu i agent zarządzania nie jest już istnieje. Agent zarządzania próbuje ponownie połączyć się z połączonego katalogu w wielu przypadkach.                                                                                                                                                                                                                                                                                                         |
| Uwierzytelnianie nie powiodło się     | Uwierzytelnianie nie jest możliwe przy użyciu podanych poświadczeń.                                                                                                                                                                                                                                                                                                                                                                                                                          |
| uprawnienia nie powiodło się         | Niewystarczające uprawnienia do kontenera w połączonego katalogu. Ten błąd jest oczekiwane tylko w przypadku agentów zarządzania dostępu protokołu LDAP (Lightweight Directory), które wyszukiwać różnych kontenerów połączonych.                                                                                                                                                                                                                                                              |
| Wyszukiwanie nie powiodło się            | Kontener lub tabeli wyszukiwania nie powiodło się z powodu nieoczekiwanego błędu.                                                                                                                                                                                                                                                                                                                                                                                                                            |
| Ostrzeżenie nie-znak wodny      | Agent zarządzania nie można odczytać znaku wodnego, podczas wykonywania pełnego importu. Ten błąd jest oczekiwane tylko w przypadku agent zarządzania dla Sun jednego katalogu Server 5.1 (dawniej inne serwer katalogowy) po ukończeniu konfiguracji agenta zarządzania początkowej i połączonego katalogu ma włączony dziennik zmian. Później gdy połączonego katalogu dziennik zmian jest wyłączona, jeśli nie zaktualizowano konfigurację agenta zarządzania, to ostrzeżenie występuje, gdy wykonywane jest pełne importowanie. |
| nie start-— operacja usuwania partycji | Partycja LDAP pierwotnie skonfigurowane dla tego agenta zarządzania nie jest już istnieje. Ten błąd jest zwracany, gdy partycja została usunięta lub partycji została usunięta i utworzona ponownie o takiej samej nazwie. W drugim przypadku nawet, jeśli nazwa jest taka sama, błąd partycji GUID będzie zostały zmienione.                                                                                                                                                                           |
